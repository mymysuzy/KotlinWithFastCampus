Intent
     - 의도, 의사

     - A야 B좀 해줘
     - B좀 해줘
     - A야 (이 데이터를 사용해서) B좀 해줘 그리고 알려줘
     - 데이터를 같이 전달할 수 있다 -> 이 번호로 전화걸어줘

     1. 명시적 Intent
        * 호출될 대상을 직접 명시하는 경우



     2. 암시적 Intent
        * 호출될 대상을 명시하지 않는 경우 -> 공유하기를 클릭하면 공유할 수 있는 목록을 쫙 보여주는 것
        * 인텐트 필터(Intent-filter)
            - 암시적 인텐트를 보낸 경우, 인텐트가 처리할 수 있는지 확인할 때 사용
            - Manifest.xml에 존재



    - 인텐트 호출 대상
        * 앱 내에서
            - 액티비티 끼리
        * 외부(안드로이드 OS, 시스템), 앱 끼리(권한이 필요)
        * 사진 앱
            - 앱 전화걸기 버튼 -> 시스템(안드로이드 OS) -> 전화걸기

    - 인텐트
        * 결과가 필요한 경우
        * 결과가 필요하지 않은 경우
