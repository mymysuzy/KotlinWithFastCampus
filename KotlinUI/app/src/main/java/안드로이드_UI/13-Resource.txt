리소스
    - drawable
        * 이미지
            - 실제 이미지 파일 혹은 xml파일
            - 동일한 이미지를 해상도별로 여러장을 준비할 수 있다
                * 핸드폰의 해상도에 맞는 이미지를 사용할 수 있다
                * 안드로이드 OS가 핸드폰 종류에 따라 적합한 해상도의 이미지를 알아서 선택해줌
    - layout
        * 화면을 구성하는 xml
    - mipmap
        * 앱 아이콘이 위치
    - values
        * colors
            - (자주 사용하는)색
            - RGB(Red, Green, Blue)
            - 16진수
        * string
            - (자주 사용하는)문자열
    - 사용방법
        * @리소스종류/리소스명
