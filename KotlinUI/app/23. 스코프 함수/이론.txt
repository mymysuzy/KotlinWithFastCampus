스코프 함수
    - 동작의 원리는 모르더라도 언제 사용하면 좋은지 알고 사용해도 가치가 있다
    - { 본문 } <- 이러한 범위를 스코프라 한다

    val victor = Person().____{
        name = "victor"
        age = 20
    }

    - apply/also/run(with)/let

    - apply/also
        * 처리가 끝나면 인스턴스(객체)를 반환

    - run(with)/let
        * 처리가 끝나면 최종값을 반환

    - apply/run(with)
        * this : 스코프 밖에 있는 변수 이름과 혼동할 수 있다

    - also/let
        * it : 스코프 밖에 있는 변수 이름과 혼동하는 것을 방지
